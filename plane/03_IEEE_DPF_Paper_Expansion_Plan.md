## 1. 섹션별 확장 상세 계획

### **I. Introduction (확장: 15줄 → 40줄 이상)**
- **연구 동기 강화:** DPF 결함의 종류(Crack vs Melting)가 실제 엔진 성능과 환경 규제에 미치는 구체적인 악영향(Euro 6/7 규제 관련 언급 등) 추가.
- **기존 연구의 한계:** 수동 검사의 통계적 한계(오검출률 구체적 수치 등)와 기존 딥러닝 방식의 데이터 의존성 문제를 더 정밀하게 비판.
- **연구의 차별성:** "Stage 0-1-2" 개념을 서론에서 더 명확히 정의하고, 도메인 브리지의 필요성을 직관적으로 설득.

### **II. Related Work (확장: 15줄 → 30줄 이상)**
- **최신 연구 동향 보강:** 2023-2024년도 소량 데이터 학습(Few-shot Learning) 관련 제조 분야 연구 사례 2-3개 추가 언급 및 본 연구와의 차이점 부각.
- **YOLO 시리즈 발전사:** YOLOv5 ~ YOLO11의 발전 과정에서 C2PSA 등 어텐션 메커니즘이 소형 객체 검출에 미치는 영향을 기술적으로 서술.

### **III. Methodology (확장: 60줄 → 120줄 이상)**
- **도메인 브리지 이론적 배경:** 
  - Proxy A-distance 관련 수식($d_{\mathcal{A}}$)을 포함한 이론적 정당성 복원 및 상세 설명.
  - X-ray 데이터의 중간 전이 역할에 대한 시각적/통계적 근거 제시.
- **YOLO11 아키텍처 심층 분석:**
  - C2PSA 모듈의 작동 원리(Spatial Attention 메커니즘)를 수식 없이도 이해 가능하도록 프로세스 상세 서술.
  - 모델의 Loss Function 구성(Box, Cls, DFL Loss)과 산업용 결함 검출에서의 의미 해석.
- **학습 프로토콜 정당화:** 
  - Stage 1, 2의 하이퍼파라미터 설정(LR, Epoch 등)에 대한 근거를 원본의 실험 노트 기반으로 보강.

### **IV. Experimental Setup (확장: 15줄 → 40줄 이상)**
- **데이터셋 통계 상세화:** 
  - 클래스 불균형 비율, 결함 크기 분포, 이미지 해상도 전처리 과정 등을 상세히 기술.
  - 데이터 증강(Augmentation) 기법(Mosaic, Mixup 등)의 구체적 파라미터와 선택 이유 설명.
- **평가 지표 정의:** 
  - 정밀도, 재현율, mAP의 수식을 다시 포함하거나, 제조 현장에서의 "False Alarm"과 "Leakage(미탐)" 관점에서 지표의 의미를 재정의.

### **V. Experimental Results (확장: 40줄 → 120줄 이상)**
- **학습 동역학(Late Blooming) 집중 분석:** 
  - **[Critical]** Epoch 1-100까지의 mAP 추이 그래프 설명 복원.
  - Phase 1~4 (적응-점진-가속-수렴) 단계별 분석을 상세히 기술 (이전 분석 노트 참조).
  - 수학적 모델링(수식 13)은 선택적으로 포함하거나 텍스트로 풀어 설명.
- **실패 사례 분석 (Failure Analysis):** 
  - 모델이 놓친 결함(False Negative)과 잘못 탐지한 경우(False Positive)의 예시를 묘사하고(이미지 설명 포함), 원인 분석(조명, 미세 균열 등) 추가.
- **클래스별 성능 상세:** Crack vs Melting의 검출 난이도 차이 분석.

### **VI. Discussion (확장: 15줄 → 40줄 이상)**
- **경제성 분석 고도화:** 투자 대비 효과(ROI) 분석을 단순 수치 나열이 아닌, 현장 도입 시나리오(장비 비용 vs 인건비 절감)로 구체화.
- **기술적 한계 및 해결책:** CPU 추론 속도(FPS) 문제에 대한 정직한 토로와 TensorRT, Quantization 등 구체적인 가속화 방안 제시.
- **확장 가능성:** 타부품(엔진 블록, 배터리 등)으로의 전이 가능성 논의.
